# VFS-LaTexViewer
「Vanilla Field Softhouse（VFS）」のブランドラインナップに加わった，直感的な数式入力・変換ツール「**VFS LaTeXViewer**」の取扱説明書を作成いたしました．

このツールは，プログラミングに近い自然な記法で数式を入力し，即座に美しいLaTeX形式への変換とプレビューを行うためのものです．

---

## VFS LaTeXViewer 取扱説明書

### 1．概要

**VFS LaTeXViewer** は，複雑なLaTeXコマンドを覚えることなく，シンプルなテキスト記法から数式を生成するためのエディタです．「バニラ（標準技術）」の軽快さを維持しつつ，リアルタイムに組版結果を確認し，変換後のソースコードをコピーしてドキュメント作成に活用できます．

### 2．各部名称と機能

#### 2.1 ヘッダー（Header）

* **VFS Logo**: Vanilla Field Softhouseの公式ロゴを配置したブランドエリアです．
* **Language Select**: 「JP（日本語）」「EN（英語）」を切り替えることができます．UI全体のテキストが即座に切り替わります．

#### 2.2 入力パネル（INPUT / LEFT PANEL）

* **Toolbar**: よく使われる数式記号のテンプレートをワンクリックで挿入できます．
* `Integral`: 定積分 ()
* `Sigma`: 総和 ()
* `Fraction`: 分数
* `Power`: べき乗


* **Editor Area (Textarea)**: ここに数式を入力します．通常の四則演算や関数（log, sinなど）を記述すると，リアルタイムで変換が開始されます．

#### 2.3 出力・プレビューパネル（OUTPUT & PREVIEW / RIGHT PANEL）

* **LATEX SOURCE**: 変換されたLaTeXのソースコードが表示されます．
* **COPYボタン**: 生成されたLaTeXコードをクリップボードにコピーします．


* **PREVIEW**: KaTeXエンジンによる高品質な数式プレビューが表示されます．白背景を採用し，実際の書類に貼り付けた際の視認性をシミュレートしています．

### 3．入力ルール（VFS記法）

VFS LaTeXViewerでは，独自のシンプルな置換エンジンにより，以下のような記法をLaTeXコマンドへ自動変換します．

| 入力内容（例） | LaTeX変換結果 | 備考 |
| --- | --- | --- |
| `(a)/(b)` | `\frac{a}{b}` | 分数（括弧で括ることを推奨） |
| `x**2` | `x^{2}` | べき乗（Python風の記法） |
| `sqrt(x)` | `\sqrt{x}` | 平方根 |
| `log(x)` | `\log\left(x\right)` | 各種関数（sin, cos, tan等） |
| `2*x` | `2x` | 係数と変数の積（アスタリスク省略可） |

### 4．便利な使い方

1. **リアルタイムプレビュー**: 入力を変更するたびにプレビューが更新されるため，構文エラー（Syntax Error）を即座に修正できます．
2. **複雑な分数のネスト**: `((a)/(b))/((c)/(d))` のように括弧を多用することで，再帰的に複雑な分数構造を構築可能です．
3. **ドキュメント作成への連携**: `COPY` ボタンで取得したコードを，OverleafなどのLaTeXエディタや，Markdownエディタの数式ブロックにそのまま貼り付けて使用してください．
